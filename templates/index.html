Abstract

This is an implementation of a new supervised learning algorithm for detecting abnormal events in confined areas like ATM room, Server room, power control room etc. The proposed method has two phases, training phase and testing phase. In the training phase, Algorithm learns the motion path and speed of objects in a video. In the testing phase if any motion happens other than in the learned the motion path or the speed of object has large variation from the learned speed then the algorithm alert it as Abnormal event.The proposed method uses dense sampling and process video in a group of frames. The algorithm uses statistical functions maximum, minimum and mode to learn the motion path and speed of objects in a video.Comparing to many other existing algorithms this algorithm has less mathematical computation involved so the processing power needed to run this algorithm is very low compared to other algorithm.
